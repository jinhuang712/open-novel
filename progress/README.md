# Progress 日志

记录每周/每个里程碑的实施进展。每次显著迭代必须在这里追加一条。

## 命名规则

```
NNN-{slug}.md
```

- `NNN`: 三位数字,从 000 起,递增
- `slug`: 简短英文标识 (init / scaffolding / agent-loop / etc.)

例:
- `000-init.md` — 项目启动
- `001-scaffolding.md` — Next.js 骨架
- `002-storage-layer.md` — 存储层
- ...

## 单条进度日志模板

```markdown
# NNN — 标题

**日期**: YYYY-MM-DD
**周次**: WX
**主要 Owner**: 自己

## 本期目标
- [ ] 任务 1
- [ ] 任务 2

## 实际完成
- ✅ 已完成 1
- ⏸ 部分完成 (50%): 详细说明
- ❌ 未完成 / 遇到阻塞: 详细说明

## 关键决策
列出本期做出的非显然技术决策与原因。

## 下期计划
- 下一步要做的事
- 已知风险

## 关联 commit
- `<sha>` 一句话描述
```
