# Archived Spec Details · 2026-06-11

本目录保存 2026-06-11 根层 spec 二层重整前的 29 篇旧 spec 原文。它们是历史材料,用于追溯字段、schema、prompt、测试和迁移细节的来源。

## 使用规则

- 需要完整旧字段、旧 prompt、旧测试矩阵时,可以从这里查原文。
- 从这里抽取仍有效的实现明细时,应迁入 `spec/appendix/` 对应分类。
- 如果抽取内容改变主路径、职责边界、失败语义或用户可见结果,必须同步更新根层 spec。
- 阶段叙事、旧方案、已关闭问题和历史 TODO 不应迁回 active appendix。
